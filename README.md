# 畅言 -> Valine

导入畅言的留言数据到 Valine 中。Valine 使用 LeanCloud 作为数据存储。

### 使用
* git clone
* npm install
* cp .env.sample .env
* vim .env
* npm run import

### 参考
* Valine: https://valine.js.org
* LeanCloud: https://leancloud.cn/docs/leanstorage-started-js.html
* 畅言评论格式说明: http://changyan.sohu.com/help/b-backup.html
